## 【打卡任务】在 deepin 25 操作系统上调通文心大模型

### 1. 任务背景

验证文心大模型在国产操作系统 **deepin 25** 上的开发兼容性。开发者需要通过 deepin 终端完成 SDK 安装、环境配置，并成功调用云端文心大模型进行对话，展示 deepin 对 AI 开发者生态的高效支持。

### 2. 准备工作

- **系统镜像**：下载并安装 [deepin 25 最新版镜像](https://www.deepin.org/zh/download/ "null")。
    
- **安装指南**：请参考下载页面的 **“deepin 25操作系统 安装指南”**。
    
- **获取 API Key**：登录 [飞桨 AI Studio 平台](https://aistudio.baidu.com/ "null") 获取个人 Access Token（用于调用云端模型）。
    

### 3. 操作步骤

#### 第一步：系统部署

在物理机或虚拟机中安装并启动 deepin 25，确保网络连接正常。

#### 第二步：环境配置

点击任务栏的终端图标，打开系统终端 **deepin-terminal**，依次输入以下命令：

1. **更新系统软件包清单**：
    
    ```
    sudo apt update
    ```
    
2. **一键安装 Python 与 飞桨 SDK**：
    
    ```
    sudo apt install -y python3-pip
    pip install erniebot --break-system-packages
    ```
    
#### 第三步：编写交互脚本

在终端输入 `nano task.py`，将以下内容粘贴进去，注意替换 `你的Token`：

```
import erniebot

# 配置 API Token
erniebot.api_type = 'aistudio'
erniebot.access_token = '在这里粘贴你的Access Token'

# 创建一次对话
response = erniebot.ChatCompletion.create(
    model='ernie-lite',
    messages=[{'role': 'user', 'content': '你好，我是 deepin 25 用户，请问 deepin 系统的主要特点是什么？'}]
)

print(f"\n[文心大模型回复]: \n{response.get_result()}")
```

按 `Ctrl+O` 保存，`Ctrl+X` 退出。

#### 第四步：运行并交互

在 **deepin-terminal** 中运行：

```
python3 task.py
```
