本仓库用于第十届飞桨黑客马拉松比赛 - deepin 系统赛题准备，赛题方向为：基于deepin系统和文心大模型实现一个智能 AI 应用。
# 任务设定
## **打卡任务** 

- **目标**：让开发者在 deepin 环境下快速跑通模型。
    
- **任务详情和提交方式**：[打卡任务](打卡任务.md)
    
- **奖励设定**：前 5 名完成者各 100 元奖金/礼品，6-10 名获得小礼品 。
## 最终任务

- **题目**：deepin Agent Teams 
    
- **任务详情和提交方式**：[最终任务](最终任务.md)
    
- **奖励**：最多 2 个名额，每人 2000 元 。

# 资源与技术准备

- **环境支持**：为认领任务的开发者提供必要的支持。deepin 镜像已经在打卡任务中说明，特定的开发环境配置指导可通过[深度论坛](https://bbs.deepin.org/)或线上进行。
    
- **直播讲解**：准备一场任务讲解直播，介绍 deepin 操作系统以及目前与文心大模型的合作基础。这里主要介绍 UOS-AI 中接入的文心模型，以及实现的一些功能。
    
- **技术支持**：负责对报名开发者进行面试，确认其能力是否匹配，并在开发过程中提供必要的答疑支持，确定最终的获胜者。
# 宣传与合规

- **渠道分发**：在 deepin 官网、公众号、开发者社区宣传本次黑客松活动。
- **成果验收**：在 6 月份对最终提交的作品进行技术评审和胜出者确定。
